## Introduction

This repository contains the necessary files for my Dockerized development
environment.

## Install

#### Pre-reqs

1. [Docker](https://docs.docker.com/install/)
1. Make
1. GitHub SSH keys for devbox in `~/.ssh/devbox`

#### Run

1. Clone repo then `cd devbox`
1. `make dotfiles`
1. `make run`
